# Contributing to Wrighty

Wrighty is currently in an early alpha stage. Its design, commands, configuration,
and internal architecture may change substantially without notice.

## Code contributions

Wrighty is not currently accepting pull requests, patches, or other code
contributions.

Please do not begin implementation work or open a pull request with the
expectation that it will be merged. This policy may change after the project
and its public interfaces have stabilized.

The MIT License permits you to fork and modify Wrighty independently.

## Bug reports and feedback

Bug reports and focused feedback are welcome through GitHub Issues.

Before opening an issue:

- check whether a similar issue already exists;
- include the Wrighty version and operating system;
- describe the expected and actual behavior;
- include a minimal reproduction when possible;
- remove credentials, tokens, private repository information, and other
  sensitive data.

Feature requests may be considered, but no roadmap or implementation commitment
is implied.

For security vulnerabilities, do not open a public issue. Follow the process in
[SECURITY.md](SECURITY.md).
