# Security Policy

Wrighty is an early-stage hobby project. Only the latest published release is
considered for security fixes, and no response or remediation timeframe is
guaranteed.

## Reporting a vulnerability

Please do not report suspected security vulnerabilities through public GitHub
issues or pull requests.

Use GitHub's private vulnerability reporting from the repository's
**Security** page by selecting **Report a vulnerability**.

Include the affected version, reproduction steps, and potential impact when
possible. Please allow time for investigation before publicly disclosing the
issue.

Wrighty does not operate a bug bounty program.
